Описание проекта

Лендинг-страница с адаптивной версткой и переключением тем оформления (тёмная / светлая / авто).
Проект демонстрирует работу с современными возможностями CSS и организацией структуры проекта.

Основная задача — реализовать страницу, которая корректно отображается на разных устройствах и поддерживает переключение между светлой и тёмной темой.

Функциональность
Адаптивная верстка для мобильных, планшетов и десктопов
Центрирование контента и резиновая верстка
Переключение тем (светлая / тёмная / авто)
Использование CSS-переменных для цветов, фонов и состояний
Использование Flexbox и Grid для расположения элементов
Подключение кастомных шрифтов (IBM Plex Mono)
Оптимизация изображений
Технологии
HTML5
CSS3
Flexbox
CSS Grid
CSS Variables
