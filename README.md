Описание проекта

Лендинг-страница с адаптивной версткой и переключением тем оформления.
Проект демонстрирует работу с современными возможностями CSS и организацией структуры проекта.

Основная задача — реализовать страницу, которая корректно отображается на разных устройствах и поддерживает переключение между светлой и тёмной темой.

Функциональность

адаптивная верстка для мобильных, планшетов и десктопов
переключение тем (светлая / тёмная / авто)
использование CSS-переменных
использование Grid и Flexbox
работа с медиазапросами
подключение кастомных шрифтов
оптимизация изображений

Технологии
HTML5
CSS3
Flexbox
CSS Grid
CSS Variables
